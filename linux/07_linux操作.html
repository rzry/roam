目录结构

bin , /usr/bin

这个里面存二进制的可执行文件 比如 ls tar mv cat

mnt

挂载点 比如安卓的目录 就是 mnt /sd

重定向命令

重定向 > ls > 1.txt 一个大于号 是覆盖 ls >> pwd 两个大于号追加

more 查看大文件

more 1.txt 空格下 b 回看

管道 ls | more

把左边命令 交给右边处理

软连接 ln

ln -s 2.txt 1ln.txt 记住 源文件一定要全路径 sudo ln -s /home/rzry/go/src/Video/api设计.txt go 硬链接就是 不加 -s 这就跟copy 一样

grep 查看文件内的内容

grep -n 查找的东西 查的路径 . -r 如果是目录 就加 -r -n 显示匹配的行号 i 不区分大小写 v 反向查找 查找不包含的哪一行

find 查找文件

find + 目录 + -name + 'txt' find ./ -name 'eli'

tar 打包 不会压缩

压缩

tar -cf tar的名字 + 要tar的文件

解压

tar -xf 解压的tar的文件 解压到别的文件夹 tar -xf 解压的tar -C 指定的文件夹

gzip 压缩打包

打包

tar cf a.tar *

压缩

gzip -r a.tar

解压

gzip -d a.tar.gz

解包

tar xf a.tar -C ~/桌面

打包+压缩

压缩

tar -czf b.tar.gz *

解压

tar zxf b.tar.gz -C ~/桌面

which 查看命令所在的位置

ubuntu 下 需要 sudo sudo which ls

chmod

chmod u+r test.txt 给文件的拥有者 user 添加 r 读的权限

  • 添加 - 删除 = 设置

读 4 写 2 执行 1 chomd 566 test.txt 可读可执行
